Difference between document object and window objects

Windows object:

  • the windows object is top most object of domain hierarchy.
  • properties of windows are:
  • closed,controllers,custom elements,document,frames and lot more..
  • windows has lots of method including alert ,prompt ,focus,confirm,open and lots more
  • screen is one of the window property that holds information of browser screen.
    •

    Document object:

    • its a property of window object.it uses for manipulation inside the document.
    • its the root node of html document
    • document objects are html,aspx,php or other documents that loaded in browser
    • it has lots of properties like anchors,applets,body charset,close,cookie and many
    • it can access with window or just document.ex:window.document or document
